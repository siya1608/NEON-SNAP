## Deployment Options

This project is ready to be deployed. Common choices:

- Vercel (recommended for Next.js): connect the repository to Vercel and deploy. Vercel will build and serve your app automatically.
- Docker: build an image using the included `Dockerfile` and run it on any container host.

Docker example:

```bash
# build image
docker build -t puzzle-hand:latest .

# run (port 3000)
docker run -p 3000:3000 --env NODE_ENV=production puzzle-hand:latest
```

CI: a GitHub Action workflow is included at `.github/workflows/ci.yml` which will lint and build on push.

If you want, I can also add a deployment workflow to publish a Docker image to GitHub Container Registry or automatically deploy to Vercel.
